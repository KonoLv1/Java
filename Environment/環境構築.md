# 環境構築

（Windows向け）注：未完成

## Javaを開発するには一般的に下記のツールが必要となる






* **テキストエディタ**



Cのコードを作成する。Visual Studio Code、terapad、eclipseなど。



## 今回は下記のツールを使う





* **Visual Studio Code**



1. Visual Studio Codeをダウンロードする



2. インストール



3. Visual Studio Codeに拡張機能をインストールする。（Ctrl + Shift + X)

4. javaを検索、**Java Extension Pack**をインストールする。

5. JDKの設定を行う（Java Development Kit)


6. VScodeで〇〇.javaというファイルを作りコード入力




```java

class Main {
  public static void main(String[] args) {
    System.out.println("Hello World 2019");
  }
}

```

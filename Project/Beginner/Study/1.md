## INFOMATION

**java.lang.Object**

全てのクラスのスーパークラスである。

**備考** 

public class UseSub {
	public static void main(String[]args) {
		Sub obj = new Sub();
	}
}

"args" と "obj"　は変数なのでぶっちゃけなんでも良い。
ただしメインの中で重複してはならない。


**【Super.java】**

public class Super { <br>
	//	コンストラクタを作る場合は下記のようにクラス名を記述する（戻り値の型なし） <br>
	public Super() { <br>
		System.out.println("Superコンストラクタ"); <br>
	} <br>
} <br>

**【Sub.java】**

// Superを継承している <br>
public class Sub extends Super { <br>
	public Sub() { <br>
		System.out.println("Subコンストラクタ"); <br>
	} <br>
} <br>

**【UseSub.java】**

public class UseSub { <br>
	public static void main(String[]args) { <br>
		Sub obj = new Sub(); <br>
	} <br>
} <br>

**【実行】**

Superコンストラクタ <br>
Subコンストラクタ <br>

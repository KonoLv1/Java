## INFOMATION

* java.lang.Object

全てのクラスのスーパークラスである。

* 

public class UseSub {
	public static void main(String[]args) {
		Sub obj = new Sub();
	}
}

"args" と "obj"　は変数なのでぶっちゃけなんでも良い。
ただしメインの中で重複してはならない。


* 【Super.java】

public class Super {
	//	コンストラクタを作る場合は下記のようにクラス名を記述する（戻り値の型なし）
	public Super() {
		System.out.println("Superコンストラクタ");
	}
}

*【Sub.java】

// Superを継承している 
public class Sub extends Super {
	public Sub() {
		System.out.println("Subコンストラクタ");
	}
}

*【UseSub.java】

public class UseSub {
	public static void main(String[]args) {
		Sub obj = new Sub();
	}
}

*【実行】

Superコンストラクタ
Subコンストラクタ

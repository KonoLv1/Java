## INFOMATION

* java.lang.Object

全てのクラスのスーパークラスである。

*【Super.java】

public class Super {
	//	コンストラクタを作る場合は下記のようにクラス名を記述する（戻り値の型なし）
	public Super() {
		System.out.println("Superコンストラクタ");
	}
}

*【Sub.java】

// Superを継承している 
public class Sub extends Super {
	public Sub() {
		System.out.println("Subコンストラクタ");
	}
}

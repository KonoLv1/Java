# Eclipse - エクリプス（イクリプス）とは？

IBMによって開発された統合開発環境で高機能ながらオープンソースでありJavaをはじめいくつかの言語に対応している。

## 開発手順

### 1.プロジェクトの作成

Project Explorerを右クリック > New > Project > Java Project > プロジェクト名の入力

### 2. パッケージ及びJavaプログラムの作成

srcフォルダを右クリック > New > Class > パッケージ名とクラス名を入力

```java
package test;

public class Test {

    public static void main(String[] args) {

        System.out.println("Hello World");

    }

}
```

### 3. 実行

メニュー > Run > Run

# メソッドとは

一連の処理の流れをまとめたもののこと。再利用できる重要な機能

* **メソッドを定義してみる**

```java

public class Method{
	public static void main(String[]args){
		//メソッド内処理
	}
}

```

２行目から４行目のブロックがメソッドブロックであり、上記はmainメソッドと呼ぶ

* **複数のメソッドを定義する**

```java

public class Method{
	public static void main(String[]args){
		//メインメソッド内処理
	}
	public static void sub(){
		//サブメソッド内処理
	}
}



```
